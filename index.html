MadPass

MadPass is a project me and some friends in college created when practicing and honing our dev skills, we wanted to take on a new challenge of using new technologies we were not entirely familliar with. This application allows a user to quickly and easily save and store login and password information for a variety of websites, accounts, apps, etc. My team used Node.js, Express.js, MongoDB / Mongoose, Crypto npm module to encrypt data, encryption algorithm AES (Advanced Encryption System) with a 256 bit encryption key, and Bcrypt. I worked on both the front and back end making sure the layout and UI was simple and clean, and assuring that redirection of the pages along with login and password management were successful.
